          Extension

          Scientific Pitch Notation
          Source: SPN.sc
          See also: Score Guideline for Notator, PitchClassSet, Notator

          Description

          This is an implementation of Scientific Pitch Notation (SPN). For the name system used in LilyPond (N.B.: the octave number follows the SPN.) and the musician-friendly naming convention, quarter tones are supported. For its own style \ez, sixteenth tones are supported.

          In scientific pitch notation, middle A at 440 Hz is defined as A4. It is identical to the middle A at 440 Hz of A4 on Roland instruments, but 1 octave higher than the middle A at 440 Hz of A3 on Yahaha instruments. A4 in scientific notation is identical to A1 or A' in Helmholtz Pitch Notation.

          References:
          • https://en.wikipedia.org/wiki/Scientific_pitch_notation

          • https://en.wikipedia.org/wiki/Helmholtz_pitch_notation

          SPN.cps(freq: 440.0, style: 'lily')

          returns the pitch name and its enharmonically-respelled pitch names as an array from the given frequency in Hz.

          NOTE: Using the following shorcut is recommended:

          Arguments:

          freq

          Frequency in Hertz.

          style

          Pitch notation style.

          • \lily

            LilyPond style note name. Octave numbers follow the scientific pitch notation.

            -> [ a4, gx4, bff4 ]

          • \mus

            Musician-friendly name.

            -> [ a♮4, g𝄪4, b𝄫4 ]

          • \ez

            A simpler naming convention than LilyPond.

            -> [ an4, a4, gS4, bF4 ]

          SPN.midi(midiPitch: 69, style: 'lily')

          returns the pitch name and its enharmonically-respelled pitch names as an array from the given MIDI pitch number.

          NOTE: Using the following shorcut is recommended:

          Arguments:

          midiPitch

          MIDI pitch number. Float is accepted to represent a microtonal change.

          style

          Please refer to the same argument of the method .cps:

          SPN.getValue(which: 'a4', mappingSwitch: 'all')

          returns the corresponding MIDI pitch number, frequency in Hz or both from the given pitch name.

          NOTE: Using the following shorcut is recommended:

          Arguments:

          which

          Pitch any style listed in the Full list of SPN with all styles:

          mappingSwitch

          decides whether to get its MIDI pitch number, its frequency in Hz, or both.

          • \all

            returns its MIDI pitch number and frequency in Hz as an event:

            -> ( 'midi': 69.0, 'cps': 440.0 )
          • \midi

            returns its MIDI pitch number:

            -> 60.0
          • \cps

            returns its frequency in Hz:

            -> 261.6255653006
